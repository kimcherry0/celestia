# 문서번호 CEL-MF-███-2077

재난기념축제 메시지 관리 계획

작성기관중앙관리청 도시홍보국
협조기관교육문화국 / 시민안정실 / 공공방송센터
작성일자2077.██.██
문서상태삭제됨
복구율52%

1. 행사 개요

재난기념축제는 대붕괴 극복과 르셰연방 재건을 기념하기 위해 매년 개최되는 공공문화행사이다.
행사 기간 동안 셀레스티아 전역에서는 빛 퍼레이드, 공중 공연, 아이돌 축하 무대, 드론 쇼, 시민 참여형 전시가 진행된다.

공식 목적은 다음과 같다.

  1. 대붕괴 극복의 역사 계승
  2. 연방 시민의 공동체 의식 강화
  3. 미래 세대의 재난 대응 의식 고취
  4. 셀레스티아 중심 체계의 안정성 홍보

2. 추모 요소 운영 지침

피해자 개인 서사, 실종자 명단, 봉인구역 잔류자 기록, 초기 대응 실패 사례 등은 행사 주제에서 제외한다.
해당 요소는 시민 정서 불안정, 체제 불신, 비인가 추모 집회로 이어질 수 있다.

추모 표현이 필요한 경우 다음 상징으로 대체한다.

  • 흰색 조명
  • 무명 꽃잎
  • 7초간의 음악 정지
  • “우리는 살아남았습니다” 문구 송출

구체적 사망자 수, 구조 실패 구역, 봉인 결정 책임자 명단은 공개하지 않는다.

3. 핵심 메시지

재난기념축제의 반복 메시지는 다음과 같다.

“우리는 무너졌으나, 셀레스티아가 우리를 다시 세웠습니다.”

“질서가 있었기에 인류는 생존했습니다.”

“관리되지 않은 자유는 다시 붕괴를 부릅니다.”

“연방의 안정은 시민 모두의 안전입니다.”

해당 문구는 공연 전광판, 공중철도 내부 광고, 학교 교육 영상, 쇼핑몰 홀로그램, 개인 단말 공지에 분산 송출한다.

4. 시민 반응 분석

축제 기간 중 시민의 체제 신뢰도는 평균 12.4% 상승하였다.
동시에 봉인구역 관련 검색량은 31.8% 감소했으며, 대붕괴 초기 책임 소재에 관한 게시물은 자동 분류 후 노출 제한 처리되었다.

비인가 추모 해시태그 ███████는 3시간 12분 만에 비가시화 조치되었다.

5. 내부 검토 의견

[부분 복원]

“기억은 통제되지 않을 경우 분노가 된다.”

“추모보다 축제가 안전하다.”

“시민은 잃어버린 사람의 이름보다, 살아남은 도시의 빛을 오래 기억해야 한다.”

6. 대체 공개 문구

“재난기념축제는 대붕괴를 극복한 시민의 용기와 연방의 재건 의지를 기념하는 평화로운 문화 행사입니다.
셀레스티아는 더 안전한 내일을 향해 나아갑니다.”
